.. py:function:: fetch_repo_tree(repo_url: str, access_token: str, branch: str = 'main', provider: str = 'github', local_repo_path: Optional[str] = None) -> List[Dict]

   Get the file tree from a repository.

   If local_repo_path is provided, reads from that local directory (for already-cloned repos).
   Otherwise, clones the repository temporarily and reads the tree (caller must clean up).

   :param repo_url: Repository URL or path (e.g., 'user/repo' or full URL).
   :type repo_url: str
   :param access_token: Authentication token.
   :type access_token: str
   :param branch: Branch name. Defaults to "main".
   :type branch: str, optional
   :param provider: Git provider ('github' or 'gitlab'). Defaults to 'github'.
   :type provider: str, optional
   :param local_repo_path: Path to a pre-cloned local repository. If provided,
                           reads from this directory instead of cloning.
   :type local_repo_path: str, optional

   :returns: List of files and directories in the repository.
   :rtype: List[Dict]

   :raises RepositoryAccessError: If clone or traversal fails.
